# Poetic Essence - 诗意本质

## Overview
Uses classical Chinese poetry to crystallize project essence. 古体诗以极简承载极大——20字可纳2000言。此技能化用、引用、品读古诗，找到项目的"眼"。

## Poetry Format Specification

| 规范 | 要求 |
|------|------|
| **风格** | 古体诗（五言/七言为主） |
| **押韵** | 尽量押韵，但不舍本逐末 |
| **行数** | 4句（绝句体）或8句（律诗体） |
| **诗源** | 仅中国古典诗词 |

**押韵原则**：意先于韵。若押韵损害表达，宁可不押。

## The Three Modes

### 1. WRITE - 诗意定调
优先直接引用契合的古诗；若无完全契合者，则化用古诗意象/句式适配场景。

### 2. READ - 借古喻今
直接引用中国古诗，原封不动，解读当下处境。

### 3. INTERPRET - 品诗得道
深度品读诗词，提取行动启示。

## Mode 1: WRITE (化用定调)

**化用原则**：
- 借古诗的骨架（句式、韵律、意象结构）
- 替换关键词适配当前场景
- 保留原诗的神韵，但意思全新

**Example**:
```
用户：我们的项目做了三个月，方向改了四次，现在又要重新开始

[poetic-essence WRITE]

AI：让我化用一句古诗来定调这个阶段...

**化用自**：沉舟侧畔千帆过，病树前头万木春（刘禹锡）

**《四次转身》**

改道侧畔千帆过，
转型前头万木春。
今日沉舟非旧舟，
明朝直挂济沧海。

---
**化用说明**：
"沉舟"→"改道"，"病树"→"转型"——保留了"新旧交替"的意象，
但说的是项目方向的调整。

**对项目的启示**：
四次改方向不是失败——是在淘汰旧的自己。
"千帆过"不是别人的成功——是你自己换过的四条船。
```

**更多化用示例**：

| 原诗 | 化用 | 场景 |
|------|------|------|
| 沉舟侧畔千帆过，病树前头万木春 | 改道侧畔千帆过，转型前头万木春 | 项目多次转型 |
| 行到水穷处，坐看云起时 | 行到用户穷，坐看需求起 | 用户调研遇到瓶颈 |
| 不识庐山真面目，只缘身在此山中 | 不识产品真面目，只缘身在代码中 | 开发者看不清产品本质 |
| 千磨万击还坚劲，任尔东西南北风 | 千次迭代还坚劲，任尔需求南北风 | 需求频繁变更 |
| 长风破浪会有时，直挂云帆济沧海 | 上线破局会有时，直挂功能济用户 | 产品迟迟不能上线 |

## Mode 2: READ (借古喻今)

**诗源**：仅中国古典诗词（唐诗宋词、诗经、禅诗等）
**原则**：原句引用，不改动一字

**Example**:
```
用户：我们的创业项目资金快用完了，但产品还没找到PMF

[poetic-essence READ]

AI：让我想一句能道出这个处境的古诗...

**"山重水复疑无路，柳暗花明又一村"**
——陆游《游山西村》

---
**解读**：
"山重水复"——每条路都像死路。
但后半句不是"找到了出路"——是"又一村"，一个全新的世界。

**对项目的启示**：
当前困境不是要"解决"的，是要"穿过"的。
PMF 往往在"山穷水尽"时才显现——因为你终于停止挣扎，开始观察。
```

## Mode 3: INTERPRET (品诗得道)

**Example**:
```
用户：团队给我一句寄语："行到水穷处，坐看云起时"，帮我品品

[poetic-essence INTERPRET]

AI：这是王维《终南别业》中的名句...

**"行到水穷处，坐看云起时"**
——王维

---
**字面解**：
走到水的尽头，坐下来看云升起。

**深层解**：
"行"是追逐，"坐"是放下。
水的尽头，正是云的起点。云从水来，但比水自由。

**对项目的启示**：
你追逐的"水"（用户？收入？增长？）可能会"穷"。
但"穷"不是失败——是形态转换的临界点。
走到尽头，然后坐下来。新的形态会自然升起。
也许这个项目的本质不是"追"，而是"等"。
```

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| WRITE 模式原创打油诗 | 化用古诗骨架，借骨换肉 |
| 化用后失去原诗神韵 | 保留原诗的意象结构和节奏感 |
| 诗太长 | 绝句4句足矣 |
| 解读太牵强 | 只用自然契合的诗，不强求 |
| 只有文艺没有行动 | 必须有"对项目的启示" |
| 用外国诗 | 仅用中国古典诗词 |
