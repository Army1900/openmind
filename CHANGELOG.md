# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-04-08

### Added
- OpenMind marketplace structure with creative-spark plugin
- 3 core innovation skills:
  - **cross-domain**: Cross-domain analogical thinking
  - **constraint-flip**: Constraint inversion and reframing
  - **deep-dream**: Memory compression via surreal dream narratives
- Python scripts for dream generation (TF-IDF, sentiment analysis)
- Specialized agents for dream weaving and memory keeping
- Plugin templates for adding new skills

### Philosophy
- Keep it minimal - only proven, practical skills
- Avoid redundancy with AI's built-in capabilities
- Focus on skills that genuinely add value
